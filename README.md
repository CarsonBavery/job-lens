# Job Lens - AI Resume Analyzer

Job Lens is a web application powered by the Google Gemini API designed to help job seekers optimize their job application process. It allows users to create optimized resumes, cover letters, and emails, and plans to offer mock job interviews in the future.

## Features

- **Resume Optimization:** Add your skills, education, and work experience to generate optimized resumes.
- **Cover Letter Generation:** Create tailored cover letters for specific job applications.
- **Email Assistance:** Generate cold emails and normalized responses for communication with recruiters.
- **Mock Interviews (Coming Soon):** Practice for job interviews with AI-driven mock interview sessions.
- **Document Editor (Future Goal):** A built-in editor to fine-tune your documents directly within the app.

## Current Status

- **Frontend:** Partially implemented (3-4 pages including Landing, Login, and Dashboard).
- **Backend:** Setup pending.

## Tech Stack

- **Frontend:** React, TypeScript, TailwindCSS, Vite
- **Backend:** Python, FastAPI (Planned), Google Gemini API

## Getting Started

### Frontend

1. Navigate to the frontend directory:
   ```bash
   cd frontend
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
